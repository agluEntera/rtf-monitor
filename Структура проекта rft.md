Структура проекта:                                                                                                                                                                                                           
                                                                                                                                                                                                                                       
  rft-monitor/                                                                                                                                                                                                                         
  ├── .env                    ← заполни JIRA_EMAIL, JIRA_API_TOKEN, TELEGRAM_*                                                                                                                                                         
  ├── monitor.py              ← основной скрипт                                                                                                                                                                                        
  ├── requirements.txt                                                                                                                                                                                                                 
  ├── setup_cron.sh           ← установка + добавление в cron                                                                                                                                                                          
  └── sql/                                                                                                                                                                                                                             
      ├── looker_current.sql  ← для Looker Studio (текущие задачи)                                                                                                                                                                     
      └── looker_history.sql  ← для Looker Studio (история + перцентили)                                                                                                                                                               
                                                                                                                                                                                                                                       
  Следующие шаги:                                                                                                                                                                                                                      
  1. Создай бота в @BotFather → /newbot → скопируй токен                                                                                                                                                                               
  2. Получи Chat ID (напиши боту /start, потом https://api.telegram.org/botТВОЙ_ТОКЕН/getUpdates)                                                                                                                                      
  3. Создай Jira API токен на https://id.atlassian.net/manage-profile/security/api-tokens                                                                                                                                              
  4. Заполни .env                                                                                                                                                                                                                      
  5. bash setup_cron.sh                                                                                                                                                                                                                
                                                                                                                                                                                                                                       
  Для Looker Studio: Resource → Manage added data sources → Add → MySQL → вставь один из SQL запросов как Custom Query.
